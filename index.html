About Me

문제의 본질을 빠르게 파악하고, 실험하며 해결하는 백엔드 개발자입니다. 단순 구현을 넘어서, 운영 환경에서 안정적으로 작동하고 배포까지 책임지는 개발을 실무에서 직접 경험했습니다.

실무에서는 레거시 개선, 성능 최적화, 인증 시스템 구현, 자동화 배포 등 다양한 문제를 주도적으로 해결하며 비즈니스에 직접 기여해왔습니다.

개인 프로젝트에서는 FastAPI + Qdrant + OpenAI 기반 RAG 챗봇을 개발하고, Docker & GitHub Actions로 자동 배포 환경까지 구성했습니다.

기술은 문제 해결의 수단이라 생각하며, 필요한 기술을 빠르게 익히고 적용할 수 있는 실행력을 갖추고 있습니다.

이제는 더 넓은 무대에서, 결과로 기여하는 개발을 이어가고자 합니다.
